# QLHS_LT

Windows Forms app for restaurant management (Guna UI 2 + 3-layer architecture).

## Getting Started
- Requirements: .NET Framework 4.7.2+, SQL Server (LocalDB or server instance).
- Open `QLTN_LT.sln` in Visual Studio 2022 and build/run.
- Update connection strings in `App.config` if needed.

## Notes
- Documentation `.md/.txt` and `.bat` helpers are kept locally only (ignored from git).
